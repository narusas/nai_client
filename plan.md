- Vue3 + TypeScript + Vite 프로젝트 초기화
- Pinia 스토어 설정
- 기본 디렉토리 구조 설정

  - stores/
  - services/
  - types/
  - views/
  - assets/

- 최상단 NAI 설정 패널

- 하단 이미지 보기 섹션
- 반응형 레이아웃 구현

## 5. Novel AI API 연동 (3일)

- NAI API 클라이언트 구현
- 인증 및 토큰 관리
- 이미지 생성 요청 처리
- 에러 처리 및 재시도 로직

## 6. 이미지 생성 및 관리 (3일)

## 7. 이미지 갤러리 및 썸네일 뷰 (2일)

- 컷별 이미지 갤러리 구현

## 8. 설정 및 환경 관리 (2일)

## 9. 데이터 저장 및 불러오기 (2일)

## 기술 스택 상세

- **프론트엔드**: Vue3, TypeScript, Vite
- **상태 관리**: Pinia
- **스타일링**: CSS/SCSS
- **백엔드 통신**: Axios
- **이미지 처리**: 네이티브 기능 및 라이브러리 검토
